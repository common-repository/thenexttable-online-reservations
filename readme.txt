== Description ==
Setting up online reservations with TheNextTable is easy! You can use this plugin with the dashboard on the <https://www.thenexttable.com> website.

== Installation ==
1. Two options:
 * Upload "thenexttable-plugin" to the "/wp-content/plugins/" directory.
 * Install the plugin through the "Plugins -> Add New" menu in WordPress.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Connect the plugin with the correct restaurant from <https://dashboard.thenexttable.com> by pasting the restaurant-key from TheNextTable.com in the TheNextTable menu in WordPress.

== Frequently Asked Questions ==
= Where can I find my restaurant access key on dashboard.thenexttable.com =
In the settings -> plugins section, you can find your access key.
